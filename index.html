How your Adobe workflows map
Segments, metrics & containers
Segments → Cohorts + inline segments. Include / exclude with is, contains, glob wildcards, and full regex (REGEXEXTRACT / REGEXREPLACE) via Derived Properties. Calculated Metrics → Formula metrics; Analysis Workspace → Data Tables.
Data model without the slots
No rigid eVar / prop / success-event slots to provision. Events & properties live in one governed Tracking Plan, captured dynamically up to generous per-project ceilings (2,000 / 2,000 / 1,000). No cap on properties per event.
Traffic KPIs & conversion
Unique Visitors → Unique users, Visits → Sessions, Hits → Events, Page Views → PageView, Bounce Rate → the Bounce Rate session metric. Any conversion = a Funnel on the events that matter.
